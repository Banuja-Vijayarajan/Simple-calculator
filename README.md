# Simple-calculator
A basic web-based calculator developed using HTML, CSS, and JavaScript, demonstrating DOM manipulation and event handling.


This is a simple calculator built using HTML, CSS, and JavaScript.

## Features
- Addition, subtraction, multiplication, and division
- Clear display functionality
- Interactive button-based input

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open the Calculator.html file in any modern web browser.
